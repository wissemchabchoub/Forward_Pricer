## 🎥 Repository Structure  <a name = "repo-struct"></a>


1. `Forward_Pricer.ipynb`: A Jupyter Notebook to launch the app
2. `requirements.txt`: Requirements file
3.  `data`: Data folder containing a CSV file of the rates
